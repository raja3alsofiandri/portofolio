## 🚀 Features

- 📱 **Responsive Design**: Mobile-friendly layout covering Home, About, Journey, Education, Organization, Achievement, Skills, Projects, Industrial Visit, and Contact.
- 🌓 **Dark / Light Mode**: Theme toggle with smooth circle-reveal transition.
- 🌐 **Bilingual Support**: Toggle between Indonesian and English content seamlessly.
- ✍️ **Dynamic Hero Title**: Typing animation effect on the main hero section.
- 📱 **Mobile Navigation**: Sidebar navigation with swipe gesture support.
- 📜 **Interactive Certificate Marquee**: Auto-scrolling, drag/swipe-able marquee with lightbox preview.
- 🤖 **"RTA Asisten" Chatbot**: Interactive chatbot featuring keyword/intent matching, typo tolerance, and short-term conversation memory.
- 🏢 **Industrial Visit Detail Pages**: Dedicated detail pages with interactive photo gallery and lightbox.
- 📬 **Quick Contact Form**: Direct contact form that opens a pre-filled Gmail compose window.
